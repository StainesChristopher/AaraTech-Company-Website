AaraTech Company Website

Project Overview

The AaraTech Company Website is a professional business website developed as part of a Web Development Internship. It showcases the company's profile, services, industries, portfolio, careers, blog, and contact information. The project also includes a backend developed using Python Flask and MySQL to store customer enquiries submitted through the contact form.

Features

* Responsive Website Design
* Home Page
* About Us Page
* Services Page
* Industries Page
* Portfolio Page
* Careers Page
* Blog Page
* Contact Page
* Contact Form with Flask Backend
* MySQL Database Integration
* SEO Optimization (Meta Tags, robots.txt, sitemap.xml)

How to Run the Project

Frontend

1. Open the project folder in Visual Studio Code.
2. Right-click index.html.
3. Select Open with Live Server.

Backend

1. Start Apache and MySQL in XAMPP.
2. Open the project folder in VS Code.
3. Open the terminal.
4. Run:

bash

python app.py


5. Open the browser and visit:

http://127.0.0.1:5000
